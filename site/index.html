Operating Modes

Five Ways to Run

$ python agent_pulse.py
Snapshot Instant point-in-time view of all active sessions and agents
$ python agent_pulse.py --live
Live Real-time auto-refreshing dashboard with sparklines and heatmaps
$ python agent_pulse.py --history
History 14-day historical stats, daily breakdowns, and trend analysis
$ python agent_pulse.py --export
Export Machine-readable JSON output for integration with other tools
$ python agent_pulse.py --compact
Compact Minimal single-line display for narrow terminals and status bars

Get Started

Up and Running in 30 Seconds

installation
# Clone the repository
$ git clone https://github.com/DUBSOpenHub/copilot-cli-agent-pulse.git
 
# Enter the directory
$ cd copilot-cli-agent-pulse
 
# Install dependencies
$ pip install -r requirements.txt
 
# Launch the live dashboard
$ python agent_pulse.py --live
